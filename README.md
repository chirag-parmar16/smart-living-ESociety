## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/smart-living-landing.git
```

2. Navigate to the project directory:
```bash
cd smart-living-landing
```

3. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

## Customization

### Colors
The color scheme can be easily modified by updating the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --text-color: #1f2937;
    --light-bg: #f3f4f6;
}
```

### Images
Replace the placeholder images with your own by updating the `src` attributes in `index.html`. All images are sourced from Unsplash for demonstration purposes.

### Content
Update the text content in `index.html` to match your specific requirements.
